# Robin Singh Portfolio

## Vision

Build a premium engineering portfolio that feels like an interactive product rather than a traditional portfolio.

The website should showcase engineering thinking, backend architecture, production systems, AI integrations, and storytelling through smooth scroll-based interactions.

The portfolio should impress engineering managers at companies like Google, Microsoft, Amazon, Stripe, Atlassian, Razorpay, Rippling, Airbnb, Uber, and similar product companies.

It should communicate:

"I build scalable production systems."

—not—

"I know Node.js."
